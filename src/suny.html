SUNYSnacks

This page provides an overview of SUNYsnacks, detailing the technologies used and including links to the GitHub repository and live site at the bottom.

Website

Highlights

SUNYsnacks is a web-based platform designed to help students discover and review campus snacks and dining options. Users can explore various food items available at their university, leave ratings and reviews, and browse community feedback to find the best snacks. The project features a responsive design, intuitive navigation, and a dynamic review system to enhance the student dining experience. Built with FastAPI on the backend and Bootstrap with Jinja2 templates on the frontend, SUNYsnacks offers a seamless and engaging interface for food enthusiasts on campus.
